Сборка проекта

Все команды выполняются из корня проекта plant_project.

1. Создание каталога сборки
mkdir build
cd build

2. Генерация проекта с помощью CMake (с подключением vcpkg)
cmake .. -DCMAKE_TOOLCHAIN_FILE=C:/vcpkg/scripts/buildsystems/vcpkg.cmake

3. Сборка проекта
cmake --build .

Запуск тестов

ctest -C Debug --verbose


После выполнения команды будет выведен результат выполнения всех тестов.
При корректной работе проекта все тесты должны завершиться со статусом PASSED.
